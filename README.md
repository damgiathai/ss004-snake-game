## 🎮 Gameplay

### Mô tả
- Điều khiển rắn để ăn thức ăn 
- Mỗi lần ăn:
  - Rắn dài ra
  - Điểm tăng
- Game kết thúc khi:
  - Đâm vào tường ❌
  - Đâm vào chính mình ❌


### ⌨️ Điều khiển

| Phím | Hành động |
|------|----------|
| W    | Lên ↑    |
| S    | Xuống ↓  |
| A    | Trái ←   |
| D    | Phải →   |


---


## ▶️ Cách chạy

### 1. Clone project

```bash
git clone https://github.com/your-username/snake-game.git
cd snake-game
```
### 2. Compile
```bash
g++ main.cpp -o snake
```
### 2. Run
```bash
./snake
```
